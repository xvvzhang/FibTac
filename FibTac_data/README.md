# Data folder placeholder

This folder is intentionally empty in the GitHub repository.

Download the validation/test archives from the project data folder and extract them here so that the following folders appear directly under `FibTac_data/`:

- `Chess_validation/`
- `Underwater_grasp_test/`
- `Liquid_test/`
- `Granularity_test/`
- `Flow_measure_test/`

For the flow-measure task, place `Flow Measure Groundtruth.xlsx` directly inside `Flow_measure_test/`. The angle-level condition folders, such as `2_8`, should also be directly inside `Flow_measure_test/`; do not keep an extra `test/` subfolder.
